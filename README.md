# flappybirdGame
flappybird 游戏中玩家控制一只小鸟，跨越由各种不同长度水管所组成的障碍。
实现方法：采用矩形与矩形的碰撞检测算法实现碰撞检测，使用本地存储技术记录游戏数据，比如分数最高分的记录等。
前端采用的vue框架，使用的Javascript语言进行编写。本人负责前端的全部工作。
作品展示：https://liusimin0206.github.io/flappybirdGame/
